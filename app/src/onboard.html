It seems like ....

    • getting started
      guide

      This tools works using data from an initial excel sheet which you have a template provided to work with. Start by renaming the .xlsx excel workbook. Name the workbook in a unique way to help you identify the generated sheet. eg INSURANCE june-july'18 .xlsx

    • initial sheet
      guidelines

      The intial workbook should have four sheets. The names of the sheets does not matter, the order of the sheets, however matters. The columns should be labled as in the given template. * important The Order of the columns must be 1. Previous month Outstanding claims 2. Intimated claim data 3. paid claims data 4. current month closing Outstanding claims data

    • First Sheet Data
      Outstanding from previous month

      This sheet should contain data from the Oustanding sheet from the previous month. It should have the following columns in the following order 1. CLASS 2. POLICY NO 3. ClAIM NO 4. INSURED NAME 5. DATE REPORTED 6. DATE OF LOSS 7. PERIOD FROM 8. PERIOD TO 9. ESTIMATE

    • intimated claims
      sheet (2)

      This sheet should contain data from Intimated claims sheet It should have the following columns in the following order 1. CLASS 2. INSURED 3. AGENCY 4. POLICY NO 5. CLAIM NO 6. DATE OF LOSS 7. DATE REPORTED 8. INTIMATION RESERVE

    • THIRD SHEET
      paid claims data

      This should contain data from paid claims excel sheet. It should contain the following columns. 1. CLASS 2. DATE OF CHEQUE 3. CHEQUE NO 4. CLAIM NO 5. POLICY HOLDER 6. UW YEAR 7. PAYEE 8. PAID AMOUNT

    • FORTH SHEET
      Outstanding from current month

      This sheet will containd data from the outstanding claims as of the end of the current month It will have the following columns. 1. INSURED NAME 2. DATE REPORTED 3. DATE OF LOSS 4. PERIOD FROM 5. PERIOD TO 6. ESTIMATE

    • THINGS TO CHECK
      on each sheet

      Check to see that all column names are in uppercase, without any trialing white spaces and make sure they are correctly spelled. Make sure you have no empty rows after pasting your data. And make sure all data from each respective column is copy pasted accurately.
